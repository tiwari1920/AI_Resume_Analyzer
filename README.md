## Installation

1. Clone the repository
```
bash
git clone
https://github.com/tiwari1920/AI_Resume_Analyzer.git
```
3. Go to project folder
```
bash
cd AI_Resume_Analyzer
```
5. Install dependencies
```
bash
pip install -r
requirements.txt
```
7. Run the server
```
bash
python manage.py runserver
```
9. Open in browser
```
http://127.0.0.1:8000/
```
